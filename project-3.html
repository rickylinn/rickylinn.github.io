BATCHING PLANT (MATERIAL HANDLING SYSTEM)

Automated material handling system for concrete batching operations, managing the transfer of raw materials such as sand and aggregates from hoppers to silos and batching units. The system ensures accurate material flow, process efficiency, and reliable batching performance.

Projects

Project Overview

As the control system vendor for the automated material handling system, I was responsible for the control logic development, HMI/SCADA graphic implementation, system integration, testing & commissioning (T&C), and operator training.

The system was designed for concrete batching plant operations, managing the automated transfer of raw materials such as sand and aggregates from storage hoppers to silos and batching units. It integrates with the main batching plant control system for accurate load cell-based weighing and batching processes, while also interfacing with an autonomous wheel loader system for automatic material replenishment to the hoppers.

The solution improves process efficiency, ensures reliable material flow, minimizes manual intervention, and provides operators with clear real-time monitoring and control visibility through the implemented graphical interface.
